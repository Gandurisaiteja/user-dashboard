# User Directory Dashboard

## Project Overview
This is a React-based User Directory Dashboard that displays user data fetched from an API. It includes search, sorting, and navigation features.

## API Used
https://jsonplaceholder.typicode.com/users

## Features
- Display users in a table
- Search users by name or email
- Sort users by name
- Click user to view detailed information
- Clean and responsive UI

## Tech Stack
- React.js
- JavaScript (ES6)
- CSS
- Fetch API
- React Router

##  Run Locally
npm install  
npm start
